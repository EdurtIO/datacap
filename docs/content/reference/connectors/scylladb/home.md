#### 介绍

ScyllaDB 是一个分布式 NoSQL 宽列数据库，适用于需要高性能和低延迟的数据密集型应用程序。

#### 环境

---

!!! note

    如果需要使用此数据源，则需要将 DataCap 服务升级到 >= `2024.01.1`

!!!

支持时间: `2024-01-12`
