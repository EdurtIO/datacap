!!! note

    本版本是一个全新的版本，完全使用了新的插件管理系统，新的 API 等各种新特性。本次更新为全新的版本，不兼容之前的版本。升级前要做好数据备份，以免数据丢失。数据库是兼容的，只要执行升级的 SQL 就可以了。

#### 主要功能

---

- 修复未登录导致国际化获取失败的问题
- 拆分插件系统为全新的模块，支持更好的插件管理
- 支持插件的在线安装，卸载等操作
- 增加通用测试模块
- 增加全新的插件商店
- 修复数据集保存时出现异常的问题
- 使用全新的 `JsonView` 注解，支持更好的数据安全及显示
